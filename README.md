# Cammus — AI-Powered Organic Growth

> Ajudamos empresas B2B a reduzir custos com tráfego pago substituindo-o por tráfego orgânico inteligente, impulsionado por AI Agents.

---

## Sobre a Cammus

A **Cammus** é uma startup de marketing B2B em fase de lançamento que usa inteligência artificial para transformar a forma como empresas conquistam e engajam clientes.

Enquanto a maioria das empresas queima budget em tráfego pago, a Cammus constrói sistemas de crescimento orgânico sustentável — automatizados por AI Agents que trabalham de forma contínua, escalável e com custo muito menor.

---

## O que está neste repositório

Este repositório contém os ativos front-end da Cammus:

| Pasta | Descrição |
|---|---|
| `index.html` | Landing page principal da Cammus |
| `Formulario Cammus/` | Formulário de captação de leads B2B |
| `design-system/` | Design system: tipografia, cores, componentes visuais |
| `assets/` | Scripts e estilos reutilizáveis |
| `docs/` | Documentação técnica e de produto |

---

## Design System

O design system da Cammus define a identidade visual usada em todos os pontos de contato com o cliente:

- **Tipografia** — fontes e hierarquia visual
- **Paleta de cores** — cores primárias, secundárias e de estado
- **Componentes** — botões, formulários, cards e demais elementos de UI
- **Guia de uso** — diretrizes para consistência entre peças

---

## Formulário de Captação

O formulário localizado em `Formulario Cammus/` é o principal canal de entrada de leads B2B.

**Funcionalidades:**
- Captura de dados de contato e perfil da empresa
- Validação de campos em tempo real
- Integração preparada para automações de CRM/marketing

---

## Como rodar localmente

Não há dependências de build. Basta abrir o arquivo HTML no navegador:

```bash
# Clone o repositório
git clone https://github.com/caetanoo/cammus-lp.git

# Acesse a pasta
cd cammus-lp

# Abra a landing page
open index.html
# ou arraste o arquivo para o navegador
```
