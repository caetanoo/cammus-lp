FAQ

Toda a paleta é derivada de apenas 3 cores base (charcoal-1, charcoal-2, off-white) usando variações de tint/shade e transparências. Zero saturação, máximo contraste.

Sim! Contraste mínimo de 4.6:1 (AA) em todos os textos, com a maioria tendo 16:1 (AAA). Focus states visíveis, semantic HTML, keyboard navigation completa.

O sistema foi projetado para ser 100% monocromático, mas inclui um azul slate muito sutil apenas para links (opcional). Semantic colors são desaturados para manter a vibe neutra.
